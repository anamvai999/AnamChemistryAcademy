
## Getting Started
1. npm install
2. npm install @reduxjs/toolkit
3. npm i -D daisyui@latest
4. npm i react-icons
5. npm install --save react-modal
6. npm i mongoose
7. npm install @tailwindcss/aspect-ratio@latest --save-dev
8. npm i react-player
